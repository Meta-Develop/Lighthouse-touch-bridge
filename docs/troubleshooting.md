# Troubleshooting

## Scope

This document will cover dependency detection, missing or swapped devices, invalid pose streams, weak motion excitation, rotation failures, translation fallback, VMT transform issues, stale overrides, reconnect behavior, and diagnostic export.

## Current Source of Truth

Failure states, fallback semantics, runtime safety requirements, and acceptance cases currently live in the [project specification](specification.md). Symptom-based remedies will be added as implementations and hardware recordings provide verified evidence.
