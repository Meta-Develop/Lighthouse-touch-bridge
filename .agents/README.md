# Lighthouse Touch Bridge .agents

This directory stores project-local agent context for Lighthouse Touch Bridge,
a SteamVR utility that combines Meta Touch controller inputs with Lighthouse
tracker poses.

Read `../AGENTS.md` first, then load the relevant files under `docs/`,
`skills/`, and `workflows/`. The complete current product specification is
`../docs/specification.md`. Run `dotnet build` and `dotnet test` from the
repository root for routine validation.
